# NaezanCraft
MineCraft Copy Naezan's Engine

- How to build using CMake
  - 
    - git clone
  ```shell
  git clone --recursive https://github.com/Meizoa/NaezanCraft.git
  ```

    - Build (Find the folder where "CMakeList.txt" exists)
  ```shell
  mkdir BuildDir
  cd BuildDir
  cmake ..
  cmake --build .
  ```
